# EduForge 商业模式与竞品分析

---

## 盈利模式

| 层级 | 内容 | 定价 |
|------|------|------|
| 开源版 | 核心引擎 + 内置教学功能 | 免费 |
| 托管版 | 免部署 SaaS 托管服务 | 年费制 |
| AI 服务 | 统一 AI 接口代理 | 按用量付费 |
| 高级插件 | 高级学情分析、家长端等 | 订阅制 |
| 插件市场 | 第三方插件分发，平台抽成 | 分成 |

---

## 竞品对比

| 维度 | 科大讯飞 | 七天学堂 | Moodle | **EduForge** |
|------|---------|---------|--------|-------------|
| 开源 | 否 | 否 | 是 | 是 |
| AI 原生 | 部分 | 否 | 否 | 是 |
| 插件化 | 否 | 否 | 是（但老旧） | 是 |
| 中国 K12 适配 | 是 | 是 | 否 | 是 |
| 部署成本 | 高 | 中 | 中 | 低 |
| 定制性 | 低 | 低 | 中 | 高 |
| 价格 | 贵 | 中 | 免费 | 免费 |

EduForge 的独特位置：开源 + AI 原生 + 中国 K12 适配，目前无直接竞品。
